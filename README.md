## ⚡ O que é o Bochecha-IA?

O **Bochecha-IA** é a inteligência artificial mais avançada e implacável integrada diretamente ao WhatsApp. Desenvolvido sob medida pelo programador **Marcos**, ele funde a capacidade de modelos neurais de ponta (Google Gemini & Anthropic Claude) a um arsenal de ferramentas automatizadas para moderação de grupos, downloads de mídias em velocidade máxima, RPG de economia e zoeira interativa.

Tudo isso rodando de forma **100% livre**, sem restrições moralistas, filtros locais de alucinação ou censura burocrática. O Bochecha responde o que pensa e entrega inteligência real.

## 🚀 Arquitetura & Recursos de Elite

### 🧠 1. Conexão Neural Pura e Sem Censura
* **Inteligência Direta das APIs**: Sem filtros de segurança chatos ou travas de alucinação. O bot opera com inteligência natural, respondendo de forma direta, sarcástica, divertida e contextual.
* **Roteamento Inteligente**: Sistema inteligente que define o melhor cérebro para cada mensagem. Claude cuida das interações de chat de alta velocidade, enquanto o Gemini Multimodal analisa imagens, vídeos e áudios com precisão científica.
* **LTM (Long Term Memory)**: Gravação e extração automática de fatos na memória do Firestore. O bot se lembra de preferências, zoeiras e conversas antigas.

### 📡 2. Central de Downloads Multimídia (yt-dlp)
* **Downloader Universal**: Baixe áudios e vídeos de qualquer link da internet (YouTube, Instagram, TikTok, Twitter/X, Pinterest, Reddit, RedTube/NSFW) com um único comando.
* **Atualização Automática**: Motor local auto-atualizável para garantir que as mudanças nas redes sociais nunca quebrem os downloads.
* **Encaminhamento Inteligente**: Comando para encaminhar áudios e mídias diretamente para outros contatos marcando os autores de forma limpa.

### 🛡️ 3. Escudo Defensivo de Grupos (Segurança Ativa)
* **Anti-Pornografia (Visão Computacional)**: Imagens NSFW enviadas por membros comuns são deletadas em menos de 1 segundo e o membro é expulso automaticamente.
* **Anti-Link & Anti-Fake**: Combate imediato contra invasões de robôs de spam e links de convites indesejados.
* **Anti-Flood & Anti-Delete**: Registro invisível que monitora o grupo e avisa quando uma mensagem foi apagada, mantendo o controle total da conversa.

### 🪙 4. RPG, Economia e Duelos no Chat
* **Sistema Financeiro**: Carteira virtual de moedas ganhas por atividade e zoeira no chat.
* **Duelos Valendo Tudo**: Desafie membros do grupo apostando seu saldo de moedas no `/duelo`.
* **Tribunal do Grupo**: Crie julgamentos para mandar membros indesejados para a forca ou perdoá-los democraticamente.

## 🛠️ Configuração e Instalação Rápida

### Pré-requisitos
* **Node.js** v18 ou superior.
* **FFmpeg** instalado na máquina (indispensável para processar áudio, vídeo e figurinhas).
* Chaves da API Gemini/Claude e Firestore cadastradas no arquivo `config.js`.

### Passos de Instalação

1. **Clonar o Repositório:**
   ```bash
   git clone https://github.com/mg5860606-ux/bochecha-ia.git
   cd bochecha-ia
   ```

2. **Instalar Dependências:**
   ```bash
   npm install
   ```

3. **Subir o Sistema:**
   * **Modo WhatsApp Ativo:**
     ```bash
     npm start
     ```
     *(Escaneie o QR Code no terminal para autenticar)*
   * **Interface Dashboard Web:**
     ```bash
     node panel/panelServer.js
     ```
     *(Acesse em http://localhost:3030 para controlar visualmente)*
   * **Modo Terminal Local (Sem WhatsApp):**
     ```bash
     npm run local
     ```

### 📱 Instalação no Termux (Android)

Para rodar o Bochecha-IA diretamente no celular via Termux, prepare o ambiente com os comandos abaixo:

```bash
pkg update && pkg upgrade -y
pkg install nodejs ffmpeg python git -y
git clone https://github.com/mg5860606-ux/bochecha-ia.git
cd bochecha-ia
npm install
npm start
```
*(O Bochecha detectará automaticamente o Termux e usará o FFmpeg instalado no sistema).*
