O **Bochecha-IA v6.0** é um organismo cognitivo de alto desempenho integrado ao WhatsApp. Projetado para operar em servidores e responder de forma direta e sem filtros, ele combina inteligência avançada (Claude, Gemini), download de mídias de qualquer site e ferramentas completas para entretenimento e administração de grupos.

---

> [!IMPORTANT]
> **DIRETRIZ DE PODER SUPREMO**
> O Bochecha obedece cegamente ao seu Criador, **Marcos**. Ele possui proteção automatizada contra remoção ou rebaixamento em grupos, além de dar prioridade absoluta a tudo que for comandado por ele.

---

## ⚡ Recursos Principais (v6.0)

### 🧠 1. Inteligência Direta e Sem Filtros
O Bochecha agora responde utilizando **100% da inteligência direta das APIs** (Gemini/Claude). Todos os filtros de alucinação locais, supressão de conversas normais de grupo e bloqueios de segurança (safety gates) foram desativados. O bot responde o que for gerado pela IA.

### 👑 2. Sistema Imperial de VIPs
* **Realeza**: O criador Marcos pode definir usuários VIPs no banco de dados. VIPs recebem tratamento real e prioritário.
* **Revelação de Ordens**: Caso questionado, o Bochecha revela de forma transparente o motivo do tratamento especial marcando o criador.

### 📡 3. Downloader Universal (yt-dlp)
* **Downloads Ilimitados**: Downloads automáticos de áudios e vídeos de qualquer rede social ou site (YouTube, TikTok, Kwai, Twitter/X, Reddit, RedTube/NSFW, etc.) através do motor local `yt-dlp`.
* **Resiliência**: Download e atualização automática do binário `yt-dlp` em segundo plano se ausente.

### 🤣 4. Reações e Figurinhas de Memes
* **Reações**: Escolha inteligente e contextual de emojis para reagir a mensagens.
* **Meme Stickers**: O bot pode responder enviando stickers convertidos na hora de memes clássicos brasileiros (Gretchen, Nazaré, Chapolin, etc.) mantendo o chat ativo e divertido.

### 📤 5. Encaminhamento Inteligente
* Marcos e Administradores podem comandar o bot para encaminhar mídias ou textos a outros chats ou contatos privados (ex: *"bochecha manda esse áudio pro Lucas"*), identificando o solicitante com links transparentes.

### 🛡️ 6. Escudo de Segurança do Grupo (Firestore Sync)
Segurança robusta rodando em thread-safe com o Cloud Firestore:
* **Anti-Pornografia (NSFW)**: Monitora envios de mídia por membros comuns. Caso a visão computacional do `api4ai` classifique a imagem com probabilidade NSFW > 60%, a mídia é apagada e o usuário expulso.
* **Anti-Link**: Remoção de links de grupos e banimento imediato de spans de WhatsApp.
* **Anti-Status**: Elimina tentativas de mensagens com travamento de status do WhatsApp.

## 🛠️ Instalação Rápida

1. **Clonar e Instalar Dependências:**
   ```bash
   git clone https://github.com/mg5860606-ux/bochecha-ia.git
   cd bochecha-ia
   npm install
   ```
2. **Executar:**
   * Modo produção (WhatsApp ativo): `npm start`
   * Modo local (Terminal): `npm run local`
